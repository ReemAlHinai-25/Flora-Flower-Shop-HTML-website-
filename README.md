
# 🌸 FLORA - Flower Shop Website

A bilingual (Arabic/English) flower shop website built with HTML, CSS, Bootstrap, and JavaScript.

## Features

- 🛍️ Shop page to browse and buy flowers
- 💐 Custom bouquet builder
- 🛒 Shopping cart
- 📞 Contact page
- ℹ️ About page
- 🧑‍💼 Owner dashboard
- 🌐 Arabic & English support with RTL/LTR switching

## Built With

- HTML5
- CSS3
- Bootstrap 5
- JavaScript

## Pages

| Page | Description |
|------|-------------|
| index.html | Home page |
| shop.html | Flower shop |
| bouquet.html | Custom bouquet builder |
| cart.js | Shopping cart |
| contact.html | Contact form |
| about.html | About the shop |
| Owner.html | Owner dashboard |
